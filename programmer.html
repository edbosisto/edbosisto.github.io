A foundation in Python inspired me to embrace computer science

Background

In 2021, I reached a crossroads in my life and career, but was fortunate to be in a situation where I had time to experiment and induldge in anything I found interesting. One of these interests led me to an online course for the programming language, Python, and I immediately fell in love with the world of computer science and programming.

For anyone interested, this is the 100 Days of Code Python Udemy course which got me started.

Since then I've accumulated a solid knowledge of the Python language and its libraries, and built numerous Python programs. Anything I think might be helpful (or cool) can be found in my Github repositories.
